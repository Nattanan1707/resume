การเขียนสะท้อนตนเองแบบการเรียนรู้ผ่านประสบการณ์ Week # 1
(Recursive) ประสบการณ์ในสัปดาห์นี้มีการเชื่อมต่อกับประสบการณ์หรือแผนการที่ผ่านมาหรือไม่? (มี/ไม่มี ทำอะไรมาบ้าง) : มี เคยฝึกงานและใช้ HTML ในการทำงาน
(CE) อะไรคือประสบการณ์ที่ผ่านมาของคุณ ? (เขียนประสบการณ์) : ได้ฝึกงานเป็น Tester บน Website และใช้ HTML ในการทำงานบน RIDE-RobotFramework
(RO) อะไรคือประสบการณ์ที่สำเร็จและล้มเหลว ? (เขียนการสะท้อนประสบการณ์ ความสำเร็จ) : สามารถเขียนโค้ดและ Test ระบบบน Website ได้ (เขียนการสะท้อนประสบการณ์ ความล้มเหลว) : บางตัวก็หา Element กับ Xpart และเป็นโค้ดที่ไม่เข้าใจจึงต้องปรึกษาพี่เลี้ยง
(AC) คุณได้เรียนรู้อะไรจากประสบการณ์ ? (เขียนการเรียนรู้ที่ได้จากประสบการณ์) : เรียนรู้เกี่ยวกับ การสร้าง Website รายละเอียดของ Website และภาษาที่ใช้ควบคุม
(AE) คุณจะได้นำสิ่งที่เรียนรู้จากประสบการณ์ ไปใช้อย่างไรต่อไป ? (เขียนสิ่งที่คุณจะทำต่อไป) : สามารถนำไปใช้ในการศึกษาต่อหรือการทำงานได้ในอนาคต และ นำความรู้ที่มีไปพัฒนาตัวเอง
